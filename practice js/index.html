there r 2 Datatypes in js: Primitive and Non-Primitive

Primitive / value type

  1. string: represent sequence of character "" or '' or `${}`
  2. number: define both integer and floating point number
  3. boolean: represent logical entity= true false
  4. undefined: represent a variable that has been declared but not yet assigned a value
    its a keyword in js
    js pull automatically undefined if not defined
  5. null: it represent intentional absence of any object value
  6. bigint
  7. symbol: represent unique identifier

Non-Primitive / Object types / reference type

  1. Object: collection of key-value pairs where values can be primitives or other objects
    • Array
  2. Function

Variable:variable is just a name of storage location

    names in 3 ways
  1. let keyword: can change the value or variables declared with let can be reassigned a new values = mutable
    block scope(accessible within the block i.e {})
    allow initialization without requiring an initial value at the time of declaration ex: let x;
  2. const keyword: cannot change the value or variables declared with const cant be reassigned after declaration
    block scope(accessible within the block i.e {})
    requires an initial value during declaration and once assigned this value can't be changed
  3. var keyword: is global scoped/ func scoped
    can be reassigned
    allow initialization without requiring an initial value at the time of declaration ex: let x;
    4. note:return value of variable is undefined for string value when typed in console
    let userName = 'Radha'
    common naming conventions for variable in js: code readability shows how code used based on their usage and scope within codebase
    • camelCase: starts with lower case letter and each new word begins with a capital letter ex:firstName, totalAmount, isLoggedIn
    • PascalCase / (UpperCamelCase): starts with uppercase letter used for mainly constructor function, classes ex:Person, CustomerOrder,EmployeeDetails
    • snake_case: lowercase with under scores _ ex: user_name, total_amount, is_logged_in
    • kebab-case: lowercase with hyphens -used in file-name,urls,css class names, data attributes in html
